Skill Levels

Every psycast is an RPG skill that levels from 1 to 10 - up to 15 with the Ascendance specialization. Learning a psycast is its first level; each further level costs a point and demands a higher psycaster level.

The level model

A psycast is always in one of three states:

  • Unlearned - shown as 0 / 10, disabled. A node in the tree you have not unlocked.
  • Learned - 1 / 10, castable. The single point you spend to unlock a psycast both learns it and moves it 0 → 1. Learning is the first level.
  • Leveled - 2 / 10 and up. Each click spends one more point, gated by the curve below.
Denominator = the cap. Normally 10, or 15 once the pawn owns the Ascendance specialization. The numerator is how far you have invested.

The requirement curve

To raise a skill to level n, the pawn's psycaster level must meet a threshold. Level 1 is free (it comes with learning); the requirement ramps from level 2 with a linear term plus a triangular term, so the last few levels are a genuine end-game investment.

Level requirement
LevelReq(n) = TierBase + (n−1)·per + (n−1)(n−2) / 2
ntarget skill level (1 = free, ramp begins at 2)
perPsycaster levels per skill level - setting, default 3
TierBase(VPE tier − 1) × per; deeper psycasts (higher VPE tier, 1-6) floor higher

Maximum reachable level

Cap
MaxLevel = max { n : LevelReq(n) ≤ psyLevel }, clamped to maxSkillLevel + capBonus
psyLevelthe pawn's current VPE psycaster level
maxSkillLevelper-ability ceiling - setting, default 10
capBonus+5 if the pawn owns the Ascendance (Synthesis) specialization, else 0

What a level does

A skill scales exactly one primary stat - its damage, area, duration, range, cost, cooldown or a count - chosen skill-aware from what the ability actually uses. Each level adds a fixed raw amount, shown on the tooltip's Each level: +N ... line.

Per-level gain
gain = baseStat · perLevelPct · primaryStrength · specBonus
baseStatthe ability's own value for its primary stat
perLevelPctthe global per-level scaling knob (setting)
primaryStrengtha small rarity weight so capstones scale a touch harder
specBonusmultipliers from Potency / Catalyst / path specializations
Level your neighbours. A skill's own levels scale only its own primary stat. To empower its other stats you level its path-mates - that is what synergies are for. Multi-skilling a whole path always beats dumping every point into one ability.

Investing - the interface

Click a learned, below-cap icon in VPE's native tab or in Modern Psycasts UI to invest one level. A plain click opens a styled confirm; Shift-click skips it. A green + marks any skill you can afford to raise, and a Roman-numeral chip shows the current level. Costs come from the same psycaster point pool VPE already uses.
