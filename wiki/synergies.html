Synergies

Path-mates empower one another. Every psycast both receives bonuses from a fixed set of neighbours and empowers others in turn - Diablo-2-style directed links, generated from your installed paths and made skill-aware so the bonus always matters.

See it live. The synergy lab on the home page lets you level Firestorm's three neighbours and watch its damage, area, and cooldown move.

Two directions

Each skill tooltip splits its synergies into two named blocks:

  • <Skill> receives bonuses from: the 3-5 path-mates that feed this skill. Each row reads +X Stat per level - the amount that source contributes per its level. Hold Alt to add the live total at the source's current level.
  • <Skill> also empowers: a single line listing the mates this skill feeds; each is fully detailed on its own card, so there is no duplicated math.

The rate

A synergy link carries a type (the stat it boosts) and a rate. The total a source contributes scales with the source's level.

Link rate & contribution
rate = SynergyRate(type) · EdgeStrength(src → dst) bonus(L) = L · rate # total at source level L rawPerLevel = baseStat(dst) · rate # what the tooltip prints
typethe boosted stat (Damage, Area, Duration, Cooldown, Cost, Charges...)
SynergyRatethe base per-level rate for that type
EdgeStrengtha per-link weight (0.6-1.4) so some neighbours matter more
Lthe source skill's current level
baseStat(dst)the receiving skill's own value for the boosted stat

Because the printed number is baseStat × rate, the bonus is shown as a real amount - +1.6 Damage per level, +0.4 Area per level, +11 minutes per level - not an abstract percentage.

Skill-aware stat assignment

A synergy type is only ever assigned to a skill that can meaningfully use it. A few percent of duration on a twelve-hour buff, or extra range on a whole-map cast, is noise - so the graph skips those types for those skills entirely and reshuffles the link onto something that matters.

TypeOffered only when…
Damagethe ability deals damage (direct or explosive)
Areathe ability has a blast radius, or is a beam
Duration0 < durationTime ≤ 10000 ticks (~4 in-game hours) - long buffs excluded
Range0 < range ≤ 60 tiles - self / global casts excluded
Targetsthe ability hits multiple targets or is a single decisive strike
Cooldown · Costalways - every cast has both (the universal fallbacks)
Why Psychic guidance never gets +seconds Duration: it lasts twelve hours, so duration is filtered out for it and its incoming links land on cost, cooldown or effect strength instead.

How the graph is built

  • Each ability's primary stat and each link's type are drawn from a rarity-weighted pool, seeded by an FNV-1a hash of the ability's defName (links seed on the source|target pair, so each link rolls its own useful type).
  • Role bias nudges the roll - offensive skills lean Damage, control leans Area, boons lean effect Strength - while the universal types keep two mates from one path rarely sharing a type.
  • The graph is frozen the first time it is built and stored in settings, so your build never reshuffles across sessions. It only regenerates when the graph version changes (for example, when the skill-aware rules above were tightened).
