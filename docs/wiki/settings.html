Settings Reference

Everything is tunable. The settings panel is organized into five tabs; every control has a mouseover explanation in-game. This is the quick map of what each knob changes, with defaults.

Skills

SettingDefaultEffect
Maximum level per ability10The per-ability ceiling (before Ascendance's +5).
Psycaster levels per skill level3The per term in the requirement curve; higher = slower leveling.
Psycaster level cap, +per tier12VPE caps the psycaster LEVEL at 30; each enlightenment tier raises that ceiling by this much (Tier 10 → ~150 = a full level-15 ascendancy). 0 disables.
Per-level scaling-Global growth knob for each skill's primary stat.
Auto psycaster statsonHeat / recovery / sensitivity grow per level; manual upgrades off. See Stats.
Remove psyfocus decayonPsyfocus no longer drains over time.
